## Git安装
[Git操作指南](https://www.runoob.com/git/git-tutorial.html)
```bash
git config -global user.name "Your Name"
it config -global user.email "Your email"
```
### 生成SSH Key
```bash
ssh-keygen -t rsa -C "邮箱地址"
```
